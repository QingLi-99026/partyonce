### 8. 后端 API 集成 ✅ 100%

**新增 API 端点** (main.py):

```python
# 3D 设计 API
POST   /api/designs              # 创建设计
GET    /api/designs              # 获取设计列表
GET    /api/designs/{id}         # 获取设计详情
PUT    /api/designs/{id}         # 更新设计
DELETE /api/designs/{id}         # 删除设计
POST   /api/designs/{id}/duplicate  # 复制设计

# 场地尺寸 API
GET    /api/venues/{id}/dimensions  # 获取场地尺寸
```

**数据库表** (designs):
```sql
- id: 主键
- user_id: 用户ID
- name: 设计名称
- description: 设计描述
- venue_type: 场地类型
- venue_width/depth/height: 场地尺寸
- theme_color: 主题颜色
- objects: JSON 对象数组
- budget: 预算
- is_public: 是否公开
- status: 状态
- created_at/updated_at: 时间戳
```

## 🚀 如何运行

### 1. 启动后端服务器
```bash
cd ~/.openclaw/workspace/projects/PartyOnce/backend
source venv/bin/activate
uvicorn main:app --reload --host 0.0.0.0 --port 8000
```

### 2. 打开 3D 设计器
```bash
# 直接打开 HTML 文件
open ~/.openclaw/workspace/projects/PartyOnce/frontend/3d-designer/index.html

# 或使用本地服务器
python -m http.server 8080 --directory ~/.openclaw/workspace/projects/PartyOnce/frontend/3d-designer
```

### 3. 访问地址
- 本地文件: `file:///.../3d-designer/index.html`
