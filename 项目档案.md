# PartyOnce 项目档案
# AI智能活动策划平台
# 创建时间: 2026-02-28
# 状态: 项目启动阶段

---

## 🎯 项目概述

**项目名称**: PartyOnce  
**定位**: AI智能活动策划平台  
**目标市场**: 澳大利亚  
**服务对象**: 婚礼、生日派对、企业年会、展会路演、商业庆典

---

## 💡 核心功能

### 用户旅程
1. 📸 拍照或选择场地
2. 🤖 AI生成活动设计方案（3个方案）
3. 🎨 进入3D场景DIY设计
4. 💰 查看实时预算
5. 🏢 选择场地
6. ✅ 一键下单执行

---

## 🧩 系统模块（9个）

| 模块 | 功能 | 复杂度 |
|------|------|--------|
| 1. 用户系统 | 注册/登录/身份识别 | ⭐⭐ |
| 2. AI策划系统 | 图片+风格+预算→3方案 | ⭐⭐⭐⭐ |
| 3. 场地系统 | 场地数据库/地图/详情/预定 | ⭐⭐⭐ |
| 4. 3D设计系统 | 模板/拖拽/旋转缩放/人物模拟 | ⭐⭐⭐⭐⭐ |
| 5. 特效模板系统 | 风格模板/灯光/花艺/舞台 | ⭐⭐⭐ |
| 6. 报价系统 | 动态计算/成本分解 | ⭐⭐⭐ |
| 7. 供应链系统 | 供应商数据库/自动分单/订单跟踪 | ⭐⭐⭐⭐ |
| 8. 支付系统 | Stripe/分阶段付款 | ⭐⭐⭐ |
| 9. 评价系统 | 用户评分/分享 | ⭐⭐ |

## 🔧 技术栈

### 前端
- Vue.js
- Three.js (3D展示)

### 后端
- Python FastAPI

### 数据库
- MySQL + Redis

### AI
- OpenAI API

### 地图
- Google Maps API

### 3D
- Blender (模板制作)
- Three.js (展示)
