## 🧪 如何运行

### 1. 安装依赖
```bash
cd ~/.openclaw/workspace/projects/PartyOnce/backend
pip install -r requirements.txt
```

### 2. 配置数据库
```bash
# 创建MySQL数据库
mysql -u root -p < ../database/schema.sql
```

### 3. 启动服务器
```bash
uvicorn main:app --reload
```

### 4. 运行测试
```bash
cd ..
./test_api.sh
```
